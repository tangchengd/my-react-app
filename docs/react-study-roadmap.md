## 目标

这份文档不是讲 React 全部知识，而是结合你现在这个项目，给你一条更适合继续做下去的学习路线。

目标是：

1. 每加一个功能，就顺便学一个 React 核心点
2. 让你的项目从“练几个小 demo”慢慢变成“有结构的学习仓库”

你现在已经具备了一个很好的起点：

1. 有路由结构

## 一个适合你的学习思路

不要追求一下子做很复杂的大功能，更适合这样：

1. 每次只加一个小页面
2. 每个页面只练一个核心知识点
3. 做完就把目录、路由、样式也整理好

这样你的项目会越来越像一个真正可维护的 React 学习仓库，而不只是零散 demo。
